#  Fitting


This module will investigate fitting data with linear regression and
chi-squared fits. There are `python` and `C++` implementations of
these.


## Original Hubble paper

### `python` version

```
python hubble.py
```


### `C++` version

```
make all
./hubble
```




## Distance modulus versus redshift of supernovae

### `python` version

```
python supernova.py
```


### `C++` version

```
make all
./supernova
```

python quake.py
